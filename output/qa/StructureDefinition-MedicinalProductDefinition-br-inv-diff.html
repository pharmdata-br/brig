Restrições

IdGrauCaminho(s)DetalhesRequisitos
mpd-br-002warningMedicinalProductDefinitionProdutos genéricos devem ter a extensão generic-reference preenchida
: extension.where(url = 'http://farmaco.maxapex.net/brig/fhir/StructureDefinition/anvisa-category' and valueCodeableConcept.coding.code = 'generic').exists() implies extension.where(url = 'http://farmaco.maxapex.net/brig/fhir/StructureDefinition/generic-reference').exists()
mpd-br-003warningMedicinalProductDefinitionRegistro ANVISA deve seguir padrão brasileiro
: identifier.where(system = 'http://anvisa.gov.br/medicamentos/registro').value.matches('^[0-9]{1}\.[0-9]{4}\.[0-9]{4}\.[0-9]{3}-[0-9]{1}$')
mpd-br-004warningMedicinalProductDefinitionMudança em indicações terapêuticas definidoras requer novo MPID
: extension.where(url = 'http://farmaco.maxapex.net/brig/fhir/StructureDefinition/therapeutic-indications-defining').exists() implies extension.where(url = 'http://farmaco.maxapex.net/brig/fhir/StructureDefinition/therapeutic-indications-defining').all(valueCodeableConcept.exists())
mpd-br-005warningMedicinalProductDefinitionMudança em status legal definidor requer novo MPID
: extension.where(url = 'http://farmaco.maxapex.net/brig/fhir/StructureDefinition/legal-status-defining').exists() implies extension.where(url = 'http://farmaco.maxapex.net/brig/fhir/StructureDefinition/legal-status-defining').valueCodeableConcept.exists()
mpd-br-006warningMedicinalProductDefinitionMotivo de mudança deve ser consistente com tipo de elemento alterado
: extension.where(url = 'http://farmaco.maxapex.net/brig/fhir/StructureDefinition/identifier-lifecycle-rules').exists() implies extension.where(url = 'http://farmaco.maxapex.net/brig/fhir/StructureDefinition/identifier-lifecycle-rules').extension.where(url = 'changeReason').valueCodeableConcept.exists()
mpd-br-007errorMedicinalProductDefinitionMPID anterior deve ser fornecido quando há mudança que requer novo MPID
: extension.where(url = 'http://farmaco.maxapex.net/brig/fhir/StructureDefinition/identifier-lifecycle-rules').extension.where(url = 'changeReason').valueCodeableConcept.coding.where(code.memberOf('http://farmaco.maxapex.net/brig/fhir/ValueSet/mpid-requiring-changes')).exists() implies extension.where(url = 'http://farmaco.maxapex.net/brig/fhir/StructureDefinition/identifier-lifecycle-rules').extension.where(url = 'previousMPID').valueString.exists()
mpd-br-008errorMedicinalProductDefinitionProduto deve ter pelo menos nome fantasia OU nome científico
: name.where(type.coding.code = 'commercial').exists() or name.where(type.coding.code = 'scientific').exists()
