Resumo

Extensão complexa: Detalhes específicos do container que são elementos definidores do PCID - mudanças requerem novo PCID

  • containerType: CodeableConcept: Tipo específico do container que é elemento definidor do PCID
  • containerQuantity: integer: Quantidade de containers por unidade de embalagem
  • containerMaterials: CodeableConcept: Materiais utilizados na composição do container
