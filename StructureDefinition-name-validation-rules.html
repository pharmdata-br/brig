Aplicar regras de validação específicas para nomes de produtos conforme regulamentação brasileira

Resumo

Extensão complexa: Regras específicas para validação de nomes de produtos brasileiros

  • requiresInventedName: boolean: Indica se o produto requer nome fantasia obrigatório
  • requiresScientificName: boolean: Indica se o produto requer DCB/DCI obrigatório
  • allowedCharacters: string: Regex para caracteres permitidos no nome
