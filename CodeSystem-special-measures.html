CodeSystem: Medidas Especiais para Produto Medicinal

Official URL: https://idmp-br.github.io/brig-idmp-brasil/CodeSystem/special-measures Version: 0.0.3
Standards status: Draft Maturity Level: 0 Computable Name: SpecialMeasures

Medidas especiais definidas para produtos medicinais, como requisitos de estudos pós-autorização, planos de gestão de risco ou restrições de prescrição

This Code system is referenced in the content logical definition of the following value sets:

  • SpecialMeasures

Este sistema de código define o seguinte código:

CódigoEcrãDefinição
post-authorization-study Estudo Pós-Autorização Obrigatoriedade de conduzir estudos adicionais após aprovação regulatória
risk-management-plan Plano de Gestão de Risco Exigência de implementar e manter plano formal de gestão de riscos
additional-monitoring Monitorização Adicional Produto sujeito a monitorização adicional intensiva
restricted-prescribing Prescrição Restrita Prescrição limitada a especialistas ou em contextos específicos
patient-registry Registro de Pacientes Obrigatório Obrigatoriedade de manter registro de pacientes em uso do medicamento
controlled-distribution Distribuição Controlada Restrições na distribuição e dispensação do produto
pregnancy-prevention Programa de Prevenção de Gravidez Requisitos específicos para prevenção de uso em gestação (ex: programa similar ao iPLEDGE)
educational-program Programa Educacional Obrigatório Exigência de programas educacionais para prescritores e/ou pacientes
