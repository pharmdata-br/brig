Tipos de material de origem para substâncias biológicas

This Code system is referenced in the content logical definition of the following value sets:

  • SourceMaterialType

Este sistema de código define o seguinte código:

CódigoEcrãDefinição
animal Animal Origem animal
plant Vegetal Origem vegetal
microbial Microbiano Origem microbiana (bactérias, fungos, leveduras)
mineral Mineral Origem mineral
synthetic Sintético Origem sintética
semi-synthetic Semi-sintético Origem semi-sintética (modificação de produto natural)
human Humano Origem humana
biotechnology Biotecnologia Produzido por biotecnologia (recombinante)
